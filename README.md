# Technical Overview

This portfolio website was built as a modern single-page application using React and React Router, with reusable components and modular CSS architecture to create a scalable, maintainable codebase. The application is deployed through GitHub Pages with an automated deployment workflow using gh-pages.


The website features a responsive, Spotify/Twitch-inspired dark UI with reusable layouts, component-based page organization, and dynamic routing between pages without full page reloads. The project emphasizes clean front-end architecture, maintainable styling practices, and modern React development principles.


## Features

* Responsive design for desktop and mobile
* Multi-page navigation using React Router
* Interactive project showcase
* Embedded resume download
* Modular component architecture
* Dark Spotify/Twitch-inspired UI
* GitHub Pages deployment
* Clean, maintainable CSS structure
