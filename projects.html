Online Shopping Analysis

After our analysis, we applied both supervised and unsupervised machine learning algorithms to derive insights from the data. I have linked the project below, which can be accessed through github. The github includes the project itself, a set of presentation slides that summarized our findings, and a video file of the presentation itself.
